# コード設計 - common 領域

`common`領域では共通して使用する CSS のコンポーネントや firebase と連携するのに関係するコードを書く。

## style 以下

背景透過で Landing と Auth 系で使用するカードデザインのインライン CSS と div の枠組みがある

## firebase 以下

### firebase.ts

firebase の init 系を実行している。

### authContext.tsx

Firebase Authencation から供給される情報と、各データベースから得られる情報を
それぞれまとめて供給している。また、データベースにアクセスするまでの間はそこで隠す。

### 各ディレクトリの\*type.ts

データベース設計に即した型が定義されている。

### userAuth

Authencation の新規ユーザー作成やログイン処理、サインアウト処理、users コレクションの
内容を変更する処理などを記述する

### scoutDb

スカウトの情報の CRUD を実装

### groupDb

同様にグループ情報の CRUD を実装
