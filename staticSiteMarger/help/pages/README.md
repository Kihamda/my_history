## このアプリでできること

- スカウト情報の検索 閲覧 更新
- 所属グループごとの権限管理
- 管理者によるメンバー招待 ロール変更

## サポートを受ける前に

1. メール認証が完了しているか確認
2. 自分が対象グループに所属しているか確認
3. 権限が `VIEW` `EDIT` `ADMIN` のどれか確認

この3つでほとんど解決します
