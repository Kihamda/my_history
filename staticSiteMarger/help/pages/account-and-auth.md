# アカウントとログイン

最終更新: 2026-02-20

## ログイン前に知っておくこと

このアプリは Firebase Authentication を使います
認証済みユーザーだけが API を利用できます
さらに メール認証済みでないと実データ画面に進めません

## 初回ログイン時のチェック

### 1 メール認証

未認証なら確認画面に誘導されます
認証後に再ログインしてください

### 2 プロフィール登録

表示名とステータスメッセージを設定します
プロフィール未作成状態では利用開始できません

## プロフィールで編集できる項目

- 表示名 `displayName`
- ステータスメッセージ `statusMessage`

## よくあるエラーと対処

### 認証エラー

- トークン期限切れのことがあります
- 一度ログアウトして再ログインすると直ることが多いです

### 招待受諾に失敗する

- すでに所属済みのグループへ重複受諾しようとしている可能性
- 期限切れや無効な招待の可能性

運用側で再招待してもらうのが最短です
